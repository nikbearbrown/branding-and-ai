Part 2: Complete the Loop - Show Your Output (25 points)

Your workflow must produce something a human can actually use or see in action. Raw JSON from an AI node is NOT an output - it's intermediate data.

What We Mean by "Output":

✅ Good Outputs (What We Want):

  • Content posted to a platform (Twitter, Reddit, blog)
  • File generated (PDF report, formatted document, spreadsheet)
  • Email/notification sent to real address
  • Data visualized (chart image, formatted table)
  • Decision + action taken (e.g., "flagged as urgent AND sent to Slack")
  • Human-readable report saved as HTML/PDF

❌ Not Outputs (Don't Submit These):

  • Raw JSON from AI node
  • Console logs showing API responses
  • Text stored in n8n variable
  • Unformatted data dumps
  • "It would create X" without actually creating X

The Test: "Could a non-technical person use this?"

If your output requires opening n8n or parsing JSON to understand it, it's not done yet.

Common Questions:

Q: "My project doesn't post to social media - what's my output?" A: Generate files! Create a PDF report, save formatted HTML, export a CSV with recommendations, create a visualization image. Make something a person can open and use.

Q: "Can I use test/sandbox accounts?" A: Yes! Test Twitter accounts, test email addresses, local files - all good. We just need to see it actually produced something beyond JSON.

Q: "What if my workflow approves/schedules for later?" A: Show the approval mechanism working and the scheduled item queued. Screenshot the scheduling interface or saved schedule file.

Q: "Do I need to build a UI?" A: No! But your output should be viewable/usable. A formatted markdown file that a human can read is fine. A blob of JSON is not.

Examples by Project Type:

  • Content Generator: Show 10 generated posts/articles adapted for different platforms (as actual files or posted content)
  • Brand Analyzer: Show analysis reports generated as PDFs with specific recommendations
  • News Aggregator: Show curated/summarized content packages saved as HTML or sent via email
  • Customer Service: Show response suggestions formatted and ready to use
  • Sentiment Analyzer: Show scored content with actionable insights in a readable report format
