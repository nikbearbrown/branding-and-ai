📅 Module 1 Overview (Week 1)

Course Introduction & Foundation Setup

Total Points: 25

Module Objectives

Establish course foundation through professional introductions, skills assessment, and essential tool setup. Students will explore the Madison Framework to identify their semester project focus while documenting their current personal brand baseline and setting clear goals for both technical and brand development tracks.

Learning Objectives

✅ Create professional introduction and establish presence in course community via Figma
✅ Assess current skills and identify three key areas for improvement
✅ Explore Madison Framework and select component for semester project
✅ Document personal brand baseline and establish measurable semester goals
✅ Set up all required tools including Figma and Adobe Creative Cloud
✅ Begin building semester-long Figma portfolio workspace

Assignment: Course Introduction & Foundation Setup (Total: 25 Points)

  1. Course Onboarding (10 points)
  2. Professional Introduction (5 points)

Task:

  • Create professional introduction including background, career aspirations, and unique perspective
  • Design introduction as visual card on Figma board (not text-only)
  • Include professional photo or avatar

Deliverable: Introduction section on Figma board

Evaluation:

  • Professional presentation and design quality (3 points)
  • Content completeness and engagement (2 points)
  1. Skills & Tools Inventory (5 points)

Task:

  • Complete self-assessment of AI tools, design/brand skills, and technical abilities
  • Create visual representation (chart, graph, or designed list) on Figma
  • Identify 3 specific areas to strengthen this semester with rationale

Deliverable: Skills inventory visualization on Figma board

Evaluation:

  • Thoroughness of assessment and visualization quality (3 points)
  • Clear identification of 3 improvement areas with reasoning (2 points)
  1. Madison Framework Exploration (5 points)

Task:

  • Explore Madison Framework GitHub repository and documentation
  • Decide what component or feature you will work on for semester project
  • Create project proposal card on Figma (150-200 words)
  • Include rationale for why this contribution is valuable

Deliverable: Madison framework project proposal on Figma board

Evaluation:

  • Clear project selection with understanding of framework (3 points)
  • Strong rationale for chosen contribution (2 points)
  1. Personal Brand Baseline & Goals (5 points)

Task:

  • Current State Assessment:
    • Screenshot current LinkedIn profile
    • Identify existing brand elements
    • Document professional narrative gaps
  • Define Semester Goals:
    • Technical Goal: Madison framework contribution
    • Brand Goal: Personal brand outcome
    • Portfolio Goal: Showcase capabilities
    • Network Goal: Connection targets

Deliverable: Baseline assessment and goals section on Figma board

Evaluation:

  • Comprehensive current state documentation (3 points)
  • Specific, measurable semester goals (2 points)
  1. Essential Tools Setup (5 points)

Task:

  • Set up Figma account and create Week 1 board
  • Download Adobe Creative Cloud (free for Northeastern students)
  • Set up additional recommended tools (Canva, ChatGPT/Claude, GitHub)
  • Document all setups with screenshots on Figma

Deliverable: Tools setup confirmation section on Figma board

Evaluation:

  • Figma board properly created and organized (2 points)
  • Adobe CC successfully installed with screenshots (2 points)
  • Additional tools documented (1 point)

Success Metrics

✅ Skills Documented: Clear baseline with identified growth areas

Single Figma Board containing:

  • Professional introduction section with visual design
  • Skills inventory visualization with 3 improvement areas
  • Madison Framework project proposal (150-200 words)
  • Personal brand baseline with LinkedIn screenshot
  • Four semester goals clearly stated
  • Tools setup screenshots and confirmations

Figma Board Standards:

  • Clear section organization with headers
  • Consistent visual styling

  • Shareable link with view/comment permissions enabled

📌 Module Outcome

Students complete Module 1 with:

  • Operational workspace on Figma that will grow throughout semester
  • Clear understanding of current skills and improvement targets
  • Selected Madison Framework project for technical track
  • Documented brand baseline for transformation tracking
  • All tools installed and ready for course work
  • Foundation set for both technical and brand development tracks
